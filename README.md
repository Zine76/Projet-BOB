Projet-BOB
Description
Ce projet contient une page web statique pour le "Plan IA — Printemps numérique — Itération 1". Il s'agit d'une interface utilisateur conçue avec HTML, CSS et potentiellement JavaScript pour présenter des informations sur un plan d'intelligence artificielle dans le contexte du Printemps numérique.

Le fichier principal est index, qui est une page HTML avec un thème sombre et une mise en page responsive.

Installation
Clonez ce dépôt :

git clone https://github.com/Zine76/Projet-BOB.git
Ouvrez le fichier index dans votre navigateur web préféré.

Utilisation
Ouvrez simplement le fichier index dans un navigateur pour voir la page.
La page est conçue pour être responsive et s'adapter aux différentes tailles d'écran.
Structure du projet
README.md : Ce fichier de documentation.
index : Le fichier HTML principal de la page web.
Technologies utilisées
HTML5
CSS3 (avec variables CSS pour le thème)
